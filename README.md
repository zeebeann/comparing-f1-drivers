# Compare F1 Drivers

A website that lets you compare current Formula 1 drivers side-by-side using live data.  
Built and designed by **Zainab Ilyas** as part of Sheridan College’s IxD course: System Design: Databases and Dynamic Data.

---

## Features

- Choose **Driver 1** and **Driver 2** from the current lineup
- See side-by-side stats and comparisons (championship place, points, etc.)  

---

## Data

- **Data Source**: <a href="https://f1api.dev/"> F1 API </a>
- **Driver Images**: <a href="https://www.formula1.com/en/drivers"> Official Formula 1 Website </a>
